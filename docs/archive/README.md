# Documentation archive

**Purpose:** Preserve superseded, duplicate, or obsolete documentation without deleting history.

## Policy

1. **Never delete sensitive or ambiguous material automatically** — move here first.
2. **Canonical copies stay in their original paths** (e.g. `docs/audit/`).
3. **Archive entries are read-only reference** — do not edit; create a new doc if content must change.
4. **Classification** follows WS-01 labels: `SAFE` · `REVIEW` · `BLOCKED` · `ARCHIVE`.
